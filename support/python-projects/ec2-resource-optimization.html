AWS EC2 Resource Optimization Analysis

Designed and implemented Python scripts leveraging AWS presigned URLs to securely analyze EC2 usage patterns across multiple AWS accounts.

Challenge

Identify and analyze unused EC2 instances across multiple AWS accounts to reduce costs and optimize resource utilization, where direct access was restricted due to security policies.

Solution

Created a secure Python solution using Boto3 to generate presigned URLs, enabling temporary, credential-restricted access to EC2 metadata and usage data without compromising security or requiring persistent credential sharing.

Technical Deep Dive

  1. Configured AWS IAM roles with least-privilege permissions for EC2 and CloudWatch access.
  2. Generated presigned URLs using Boto3 S3 client to securely share temporary access tokens (1-hour expiration).
  3. Parsed EC2 instance metadata and CloudWatch metrics to analyze CPU, network, and storage utilization.
  4. Used Pandas to aggregate and analyze usage patterns, identifying idle and underutilized instances.
  5. Generated automated reports with cost-saving recommendations based on utilization thresholds.
  6. Implemented logging and monitoring via CloudWatch to track script execution and data access.

Tools & Technologies

Code Snippet Example

Python script demonstrating presigned URL generation and EC2 data retrieval.

import boto3
from datetime import datetime, timedelta
import pandas as pd

# Initialize AWS clients
s3_client = boto3.client('s3')
ec2_client = boto3.client('ec2')
cloudwatch = boto3.client('cloudwatch')

def generate_presigned_url(bucket_name, object_name, expiration=3600):
    """Generate a presigned URL for temporary, secure access."""
    try:
        response = s3_client.generate_presigned_url(
            'get_object',
            Params={'Bucket': bucket_name, 'Key': object_name},
            ExpiresIn=expiration
        )
        return response
    except Exception as e:
        print(f"Error generating presigned URL: {e}")
        return None

def analyze_ec2_utilization():
    """Analyze EC2 instance utilization from CloudWatch metrics."""
    instances = ec2_client.describe_instances()
    utilization_data = []
    
    for reservation in instances['Reservations']:
        for instance in reservation['Instances']:
            instance_id = instance['InstanceId']
            instance_type = instance['InstanceType']
            
            # Fetch average CPU utilization from last 7 days
            metric_stats = cloudwatch.get_metric_statistics(
                Namespace='AWS/EC2',
                MetricName='CPUUtilization',
                Dimensions=[{'Name': 'InstanceId', 'Value': instance_id}],
                StartTime=datetime.utcnow() - timedelta(days=7),
                EndTime=datetime.utcnow(),
                Period=3600,
                Statistics=['Average']
            )
            
            avg_cpu = sum(d['Average'] for d in metric_stats['Datapoints']) / len(metric_stats['Datapoints']) if metric_stats['Datapoints'] else 0
            
            utilization_data.append({
                'InstanceId': instance_id,
                'Type': instance_type,
                'AvgCPU': avg_cpu,
                'State': instance['State']['Name'],
                'LaunchTime': instance['LaunchTime']
            })
    
    return pd.DataFrame(utilization_data)

# Main execution
if __name__ == "__main__":
    df_utilization = analyze_ec2_utilization()
    
    # Identify underutilized instances (avg CPU < 5%)
    underutilized = df_utilization[df_utilization['AvgCPU'] < 5]
    
    print(f"Total Instances: {len(df_utilization)}")
    print(f"Underutilized Instances: {len(underutilized)}")
    print(underutilized.to_string())
